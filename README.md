
🧙‍♂️ WizardWar

WizardWar é um jogo 2D criado como um trabalho para a disciplina de Introdução à Ciência da Computação, na Uni-FACEF. O jogo coloca o jogador no controle de um poderoso mago em uma batalha épica contra goblins e um boss gigante!

🧩 Sobre o Jogo

Em WizardWar, o jogador é desafiado a sobreviver em uma arena mágica, utilizando seus reflexos e feitiços para derrotar inimigos.
O objetivo é claro: derrotar todos os goblins e o boss utilizando bolas de fogo e esferas de plasma.

✨ Mecânicas principais:

    Luta contra goblins em tempo real

    Enfrentamento de um chefe gigante

    Uso de magias com diferentes efeitos

    Sistema de pontuação integrado com banco de dados

⚙️ Tecnologias Utilizadas

    🎮 Engine: Construct 3

    ☁️ Banco de Dados: Firebase

🎮 Controles

    Setas do teclado: movimentação do mago

    Mouse esquerdo: lança bolas de fogo

    Mouse direito: lança esferas de plasma

    Mira: baseada no cursor do mouse
